Feel free to use this file in any way you need, you can edit a copy of this file as much as you want for non-commercial or commercial purposes. It's not necessary, but if you want, you can give me credit/attribution on Twitter and/or using this file. Would be just nice to see what you did with it.

A note about typefaces

This file is using platform-native typefaces for iOS & macOS, designed by Apple.

To make fonts clickable, you can download the typefaces at https://developer.apple.com/fonts/

Fonts used on the live site
